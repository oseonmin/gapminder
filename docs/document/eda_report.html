3 이상치 탐지 (IQR 규칙)

코드 보기 
do.call(rbind, lapply(c("lifeExp","gdpPercap","pop"), function(col){
  q <- quantile(df[[col]], c(.25,.75)); iqr <- diff(q)
  out <- df[df[[col]] < q[1]-1.5*iqr | df[[col]] > q[2]+1.5*iqr, ]
  top <- if(nrow(out)) paste(head(sprintf("%s(%d)", out$country[order(-out[[col]])], out$year[order(-out[[col]])]),3), collapse=", ") else "-"
  data.frame(변수=col, 이상치수=nrow(out), 상위사례=top)
})) |> knitr::kable()

4 평균의 함정: 단순 vs 인구가중

국가 단순평균은 인구 30만 소국과 13억 대국을 동일 취급한다. 인구가중 평균으로 보정한다.

중요핵심

2007년 기대수명은 단순 67세 vs 인구가중 68.9세다. 인구 많은 국가(중국·인도 등)가 다수 소국보다 수명이 높아 가중평균이 더 높다. 어떤 평균을 쓰느냐에 따라 결론이 달라지므로 가중 여부를 반드시 명시해야 한다.

힌트회귀 해석
  • 기대수명 = -9.1 + 8.4 × ln(1인당 GDP), R² = 0.652
  • 대륙·연도 통제 시 R² = 0.798
  • 소득이 10배 늘면 기대수명은 약 19.4세 증가 — 저소득 구간에서 효과가 크고(가파른 좌측) 고소득에서 둔화된다(수확체감).

10 종합 결론

  1. 분포는 로그정규 — pop·GDP는 강한 우편향. 산술평균·선형분석은 부적절, 로그척도가 표준.
  2. 평균의 함정 — 단순 국가평균과 인구가중평균은 결론이 다르다. 가중 여부를 반드시 명시.
  3. 전반적 향상 + 국지적 후퇴 — 세계 평균은 개선됐으나 짐바브웨·스와질란드 등은 기대수명이 감소.
  4. 수명은 수렴, 소득은 발산 — 기대수명 격차는 좁혀지나 소득 격차는 벌어진다.
  5. 소득–수명은 로그 선형(R²=0.65) — 빈곤국의 소득 증가가 수명 개선에 특히 효과적.
  6. 성장의 명암 — 동아시아(한국·대만·싱가포르)는 고성장, 일부 아프리카·분쟁국은 마이너스 성장.

# 분포 형태: 왜도와 첨도

요약통계가 숨기는 비대칭성을 먼저 확인한다.

```{r}
vars <- c("pop","lifeExp","gdpPercap")
data.frame(
  변수 = vars,
  평균 = sapply(vars, \(c) mean(df[[c]])),
  중앙값 = sapply(vars, \(c) median(df[[c]])),
  왜도 = round(sapply(vars, \(c) skewness(df[[c]])), 3),
  초과첨도 = round(sapply(vars, \(c) kurtosis(df[[c]])), 3)
) |> knitr::kable(format.args = list(big.mark = ","))
```

```{r}
#| fig-cap: "gdpPercap 원척도 vs 로그척도 — 강한 우편향이 로그변환으로 정규화된다"
g <- bind_rows(
  transmute(df, 값 = gdpPercap, 척도 = "원척도 (gdpPercap)"),
  transmute(df, 값 = log10(gdpPercap), 척도 = "로그척도 (log10)"))
ggplot(g, aes(값, fill = 척도)) +
  geom_histogram(bins = 38, color = "white", alpha = .9) +
  facet_wrap(~척도, scales = "free") +
  scale_fill_manual(values = c("원척도 (gdpPercap)" = "#E45756", "로그척도 (log10)" = "#4C78A8")) +
  labs(x = NULL, y = "빈도") + theme(legend.position = "none")
```

::: {.callout-tip title="해석"}
`pop`·`gdpPercap`은 강한 우편향(로그정규에 가깝다). 로그변환 후 왜도는
`log(pop)` = `r round(skewness(log(df$pop)),2)`, `log(gdpPercap)` = `r round(skewness(log(df$gdpPercap)),2)`로 거의 0이 된다.
→ **평균보다 중앙값·로그척도**가 적절하다.
:::

# 이상치 탐지 (IQR 규칙)

```{r}
#| code-fold: true
do.call(rbind, lapply(c("lifeExp","gdpPercap","pop"), function(col){
  q <- quantile(df[[col]], c(.25,.75)); iqr <- diff(q)
  out <- df[df[[col]] < q[1]-1.5*iqr | df[[col]] > q[2]+1.5*iqr, ]
  top <- if(nrow(out)) paste(head(sprintf("%s(%d)", out$country[order(-out[[col]])], out$year[order(-out[[col]])]),3), collapse=", ") else "-"
  data.frame(변수=col, 이상치수=nrow(out), 상위사례=top)
})) |> knitr::kable()
```

이상치는 오류가 아니라 **구조적 특수성**(산유국 고소득, 거대 인구국)이므로 제거가 아닌 맥락으로 해석한다.

# 평균의 함정: 단순 vs 인구가중 {#sec-weighted}

국가 단순평균은 인구 30만 소국과 13억 대국을 동일 취급한다. 인구가중 평균으로 보정한다.

```{r}
#| fig-cap: "전세계 평균 기대수명 — 단순평균(회색) vs 인구가중(빨강)"
comp <- df |> group_by(year) |>
  summarise(단순 = mean(lifeExp), 가중 = weighted.mean(lifeExp, pop), .groups="drop")
cl <- comp |> pivot_longer(-year, names_to="구분", values_to="lifeExp") |>
  mutate(text = sprintf("%d년 · %s<br>%.1f세", year, 구분, lifeExp))
p <- ggplot(cl, aes(year, lifeExp, color = 구분, group = 구분, text = text)) +
  geom_line(linewidth = 1.1) + geom_point(size = 1.6) +
  scale_color_manual(values = c(단순 = "#9aa0a6", 가중 = "#C0392B")) +
  labs(x = NULL, y = "기대수명 (세)")
gg2(p)
```

::: {.callout-important title="핵심"}
2007년 기대수명은 단순 `r round(tail(comp$단순,1),1)`세 vs 인구가중 `r round(tail(comp$가중,1),1)`세다.
인구 많은 국가(중국·인도 등)가 다수 소국보다 수명이 높아 **가중평균이 더 높다**.
어떤 평균을 쓰느냐에 따라 결론이 달라지므로 가중 여부를 반드시 명시해야 한다.
:::

# 수렴과 발산

```{r}
#| fig-cap: "기대수명은 수렴(국가간 표준편차↓), 소득은 발산(빈부격차 배수↑)"
disp <- df |> group_by(year) |>
  summarise(`기대수명 표준편차(세)` = sd(lifeExp),
            `GDP 최고/최저 배수` = max(gdpPercap)/min(gdpPercap), .groups="drop") |>
  pivot_longer(-year, names_to="지표", values_to="값")
ggplot(disp, aes(year, 값, color = 지표)) +
  geom_line(linewidth = 1.1) + geom_point(size = 1.6) +
  facet_wrap(~지표, scales = "free_y") +
  scale_color_manual(values = c("기대수명 표준편차(세)"="#2E7D32","GDP 최고/최저 배수"="#C0392B")) +
  labs(x = NULL, y = NULL) + theme(legend.position = "none")
```

기대수명은 국가 간 격차가 좁혀지는(수렴) 반면, 소득 격차는 후반부에 벌어지는(발산) **비대칭** 양상이다.

# 정량 모델: 소득과 수명

```{r}
#| fig-cap: "lifeExp ~ log(gdpPercap), 전체 연도 · 회귀선 포함"
dd <- df |> mutate(text = sprintf("<b>%s</b> %d<br>$%s · %.1f세",
                                  country, year, comma(round(gdpPercap)), lifeExp))
p <- ggplot(dd, aes(gdpPercap, lifeExp)) +
  geom_point(aes(color = continent, text = text), alpha = .45, size = 1.3) +
  geom_smooth(method = "lm", formula = y ~ x, se = TRUE, color = "#222", linewidth = 1) +
  scale_x_log10(labels = label_dollar(scale_cut = cut_short_scale())) +
  scale_color_manual(values = cont_pal) +
  labs(x = "1인당 GDP (로그 스케일)", y = "기대수명 (세)")
gg2(p)
```

::: {.callout-tip title="회귀 해석"}
- **기대수명 = `r round(coef(m1)[1],1)` + `r round(coef(m1)[2],1)` × ln(1인당 GDP)**, **R² = `r round(r2,3)`**
- 대륙·연도 통제 시 R² = `r round(summary(lm(lifeExp~log(gdpPercap)+continent+year, df))$r.squared,3)`
- 소득이 **10배** 늘면 기대수명은 약 **`r round(coef(m1)[2]*log(10),1)`세** 증가 — 저소득 구간에서 효과가 크고(가파른 좌측) 고소득에서 둔화된다(수확체감).
:::

# 종합 결론

1. **분포는 로그정규** — pop·GDP는 강한 우편향. 산술평균·선형분석은 부적절, 로그척도가 표준.
2. **평균의 함정** — 단순 국가평균과 인구가중평균은 결론이 다르다. 가중 여부를 반드시 명시.
3. **전반적 향상 + 국지적 후퇴** — 세계 평균은 개선됐으나 짐바브웨·스와질란드 등은 기대수명이 감소.
4. **수명은 수렴, 소득은 발산** — 기대수명 격차는 좁혀지나 소득 격차는 벌어진다.
5. **소득–수명은 로그 선형(R²=`r round(r2,2)`)** — 빈곤국의 소득 증가가 수명 개선에 특히 효과적.
6. **성장의 명암** — 동아시아(한국·대만·싱가포르)는 고성장, 일부 아프리카·분쟁국은 마이너스 성장.
