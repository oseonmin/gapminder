3 이상치 탐지 (IQR 1.5배 규칙)

코드 
out_tbl <- do.call(rbind, lapply(c("lifeExp", "gdpPercap", "pop"), function(col) {
  q <- quantile(df[[col]], c(.25, .75)); iqr <- diff(q)
  lo <- q[1] - 1.5 * iqr; hi <- q[2] + 1.5 * iqr
  out <- df[df[[col]] < lo | df[[col]] > hi, ]
  top <- if (nrow(out)) paste(utils::head(sprintf("%s(%d)",
            out$country[order(-out[[col]])], out$year[order(-out[[col]])]), 3), collapse = ", ") else "-"
  data.frame(변수 = col, 이상치수 = nrow(out), 상위사례 = top)
}))

4 단순평균 vs 인구가중 평균 ⭐

국가 단순평균은 인구 30만 소국과 13억 대국을 동일 취급한다. 인구가중 평균으로 보정한다.

7 정량 모델 — 회귀분석

코드 
m1 <- lm(lifeExp ~ log(gdpPercap), data = df)
m2 <- lm(lifeExp ~ log(gdpPercap) + continent + year, data = df)
s1 <- summary(m1)
  • 단순 모형: lifeExp = -9.1 + 8.41 × log(gdpPercap), R² = 0.652
  • 통제 모형(+ continent + year): R² = 0.798
  • 상관계수: 선형 0.584 → 로그변환 후 0.808
코드

소득과 수명은 단순 선형이 아니라 로그 선형 관계다. 저소득 구간에서 소득 증가가 수명을 크게 끌어올리고, 고소득 구간에서 둔화된다(수확체감).

9 종합 결론

  1. 분포는 로그정규 — pop·GDP는 강한 우편향. 산술평균·선형분석은 부적절, 로그척도가 표준.
  2. 평균의 함정 — 단순 국가평균과 인구가중평균은 결론이 다르다. 가중 여부를 반드시 명시해야 한다.
  3. 전반적 향상 + 국지적 후퇴 — 세계 평균은 개선됐으나 짐바브웨·스와질란드 등은 기대수명이 감소했다.
  4. 수명은 수렴, 소득은 발산 — 국가 간 기대수명 격차는 좁혀지는 반면 소득 격차는 벌어진다.
  5. 소득–수명은 로그 선형 (R²=0.65, 통제 시 0.80) — 빈곤국의 소득 증가가 수명 개선에 특히 효과적.
  6. 성장의 명암 — 동아시아(한국·대만·싱가포르)는 고성장, 일부 아프리카·분쟁국은 마이너스 성장.

```{r setup}
kurtosis <- function(x) { x <- x[!is.na(x)]; n <- length(x); m <- mean(x); s <- sd(x); (sum((x - m)^4) / n) / s^4 - 3 }
cagr     <- function(v0, v1, years) (v1 / v0)^(1 / years) - 1
```

```{r outliers}
out_tbl <- do.call(rbind, lapply(c("lifeExp", "gdpPercap", "pop"), function(col) {
  q <- quantile(df[[col]], c(.25, .75)); iqr <- diff(q)
  lo <- q[1] - 1.5 * iqr; hi <- q[2] + 1.5 * iqr
  out <- df[df[[col]] < lo | df[[col]] > hi, ]
  top <- if (nrow(out)) paste(utils::head(sprintf("%s(%d)",
            out$country[order(-out[[col]])], out$year[order(-out[[col]])]), 3), collapse = ", ") else "-"

}))
```

# 단순평균 vs 인구가중 평균 ⭐

국가 단순평균은 인구 30만 소국과 13억 대국을 동일 취급한다. 인구가중 평균으로 보정한다.

# 정량 모델 — 회귀분석

```{r regression}
m1 <- lm(lifeExp ~ log(gdpPercap), data = df)
m2 <- lm(lifeExp ~ log(gdpPercap) + continent + year, data = df)
s1 <- summary(m1)
```

- 단순 모형: `lifeExp = `r round(coef(m1)[1],2)` + `r round(coef(m1)[2],2)` × log(gdpPercap)`, **R² = `r round(s1$r.squared,3)`**
- 통제 모형(`+ continent + year`): **R² = `r round(summary(m2)$r.squared,3)`**
- 상관계수: 선형 `r round(cor(df$lifeExp, df$gdpPercap),3)` → 로그변환 후 `r round(cor(df$lifeExp, log(df$gdpPercap)),3)`

```{r regression-plot}
#| fig-cap: "lifeExp vs log(gdpPercap) + 회귀선"
plot(df$gdpPercap, df$lifeExp, log = "x",
     col = adjustcolor(cont_col[as.character(df$continent)], 0.5), pch = 19, cex = 0.6,
     xlab = "gdpPercap (log)", ylab = "lifeExp",
     main = sprintf("lifeExp ~ log(GDP) | R^2=%.3f", s1$r.squared))
xs <- seq(min(df$gdpPercap), max(df$gdpPercap), length.out = 200)
lines(xs, predict(m1, newdata = data.frame(gdpPercap = xs)), col = "black", lwd = 2)
legend("bottomright", legend = cont_levels, col = cont_col, pch = 19, bty = "n")
```

소득과 수명은 단순 선형이 아니라 **로그 선형 관계**다. 저소득 구간에서 소득 증가가 수명을 크게 끌어올리고, 고소득 구간에서 둔화된다(수확체감).

# 종합 결론

1. **분포는 로그정규** — pop·GDP는 강한 우편향. 산술평균·선형분석은 부적절, 로그척도가 표준.
2. **평균의 함정** — 단순 국가평균과 인구가중평균은 결론이 다르다. 가중 여부를 반드시 명시해야 한다.
3. **전반적 향상 + 국지적 후퇴** — 세계 평균은 개선됐으나 짐바브웨·스와질란드 등은 기대수명이 *감소*했다.
4. **수명은 수렴, 소득은 발산** — 국가 간 기대수명 격차는 좁혀지는 반면 소득 격차는 벌어진다.
5. **소득–수명은 로그 선형 (R²=0.65, 통제 시 0.80)** — 빈곤국의 소득 증가가 수명 개선에 특히 효과적.
6. **성장의 명암** — 동아시아(한국·대만·싱가포르)는 고성장, 일부 아프리카·분쟁국은 마이너스 성장.
